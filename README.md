# Nimbus Weather App

Nimbus is a modern, responsive weather web application built using JavaScript and Visual Crossing Weather API. It provides current weather conditions, detailed information, and a 7-day forecast. The app features dynamic backgrounds and temperature unit toggle functionality.

## Features

- Display current weather for any city
- Shows detailed weather information: UV index, humidity, wind, pressure, visibility, sunrise, sunset, feels like temperature
- 7-day forecast with weather icons
- Dynamic background that changes according to weather conditions
- Temperature unit toggle between Celsius and Fahrenheit
- Responsive and dark-themed UI
